## 🚀 Visão Geral
Este framework representa o estado da arte em automação de testes End-to-End (E2E). Desenvolvido por **Luciana Orli**, o projeto foca em alta performance, escalabilidade e observabilidade, utilizando o ecossistema **Python + Playwright + BDD**. 

## 🛠️ Stack Tecnológica & Diferenciais
* **Engine:** Playwright para automação de browser ultra-rápida.
* **Metodologia:** BDD (Behavior Driven Development) com Pytest-BDD.
* **Reporting:** Allure Framework para dashboards e análise de causa raiz.

## 🏗️ Engenharia de Software (SDET Focus)
* **Page Object Model (POM):** Arquitetura desacoplada para manutenção ágil.
* **Gherkin Syntax:** Documentação viva e requisitos técnicos.
